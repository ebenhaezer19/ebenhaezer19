## About Me

I am an IT & Big Data Analytics student with strong interest in software engineering, system design, and cybersecurity analysis (SOC Level 3). My work focuses on building practical, reliable, and scalable applications using modern web technologies and data-driven workflows.

I enjoy designing backend architectures, implementing secure transaction flows, and exploring threat detection patterns. Most of my projects involve:

Web development with Python (Django/Flask), JavaScript, PHP, and PostgreSQL

System design and multithreading concepts

Data modeling and analytics

Security operations, log analysis, and detection engineering

I constantly explore new technologies and strive to build software that is robust, maintainable, and aligned with real-world operational needs.
